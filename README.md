# 🧮 Calculator

A simple calculator made using **HTML**, **CSS**, and **JavaScript**. This project can perform basic arithmetic operations with a clean and responsive design.

## 🚀 Features

- Addition
- Subtraction
- Multiplication
- Division
- Clear (AC)
- Delete (DE)
- Responsive Design

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 📂 How to Use

1. Download or clone this repository.
2. Open the `index.html` file in your browser.
3. Start calculating!

I built this project to practice my HTML, CSS, and JavaScript skills and improve my understanding of front-end development.
